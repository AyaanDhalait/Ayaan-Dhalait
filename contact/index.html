Fun Day Before Monday

Fun Day Before Monday is an interactive activity generator built with Python and Streamlit. It helps users plan fun and productive activities for the weekend or before the start of the week. The app randomly suggests games, creative tasks, or mini challenges to make the day enjoyable.

Features:

  • Random activity suggestions
  • User-friendly web interface using Streamlit
  • Categories for creativity, outdoor fun, and mental challenges

Tech Used: Python, Streamlit

DHEngine

DHEngine is a lightweight Python game engine built on top of Pygame. It simplifies the process of creating 2D games by providing ready-to-use functions, classes, and tools. Ideal for indie developers or students learning game development.

Features:

  • Easy-to-use game loop and event handling
  • Tile mapping and sprite management
  • Custom physics and collision detection
  • Extensive documentation and examples

Tech Used: Python, Pygame
